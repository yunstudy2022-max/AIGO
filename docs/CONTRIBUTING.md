# Contributing to AIGO

Thank you for your interest in contributing! Here's how to get started.

## Development Setup

```bash
git clone https://github.com/yunstudy2022-max/AIGO.git
cd AIGO
python -m pytest tests/ -v   # make sure tests pass
```

## How to Contribute

- **Bug reports** — open an issue with steps to reproduce
- **Feature requests** — describe the use case in an issue first
- **Pull requests** — please include tests for new utilities

## Code Style

- Follow PEP 8
- Add docstrings to all public functions
- Keep functions small and focused

## Questions?

Open an issue and tag it `question`.
